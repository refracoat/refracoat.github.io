# Brand Design

## Logo

The Refracoat logo consists of:
- **Graphic Element**: A stylized C-shaped curve with an abstract flame/leaf/wing element in gold
- **Primary Text**: "REFRACOAT" in bold, uppercase, gold sans-serif
- **Secondary Text**: "SOLUTIONS" in smaller, uppercase, gold sans-serif below the primary text

The logo should be displayed in gold/yellow tones that match the brand color palette. The logo file should be stored in `/public/logo.svg` (or `/public/logo.png`).

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Primary Yellow | `#f3d34a` | Main brand color, logo colors |
| Light Yellow | `#f3e37c` | Secondary accents |
| Pale Yellow | `#f2e5a2` | Backgrounds, subtle highlights |

## Typography

### Headings
**Bebas Neue** - Bold, condensed sans-serif for headlines and display text.

### Body
**Montserrat** - Clean, versatile sans-serif for body copy and UI elements.
