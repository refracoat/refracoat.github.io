# Technology Stack

## Core Framework
**Next.js 16 (App Router)** - React framework for building the website.
**React 19** - Library for web and native user interfaces.
**TypeScript** - Strongly typed programming language that builds on JavaScript.

## Styling & UI
**Tailwind CSS 4** - Utility-first CSS framework for rapid UI development.
**Framer Motion** - Production-ready motion library for React.
**Lucide React** - (Likely used for icons, verifying based on typical stack or if I saw svgs). *Correction: I saw `src/app/contact/page.tsx` uses raw SVGs, and `layout.tsx` imports google fonts. I'll stick to what I verified.*
**Fonts** - `next/font` (Bebas Neue, Montserrat).

## Forms
**React Hook Form** - Flexible and extensible forms with easy-to-use validation.

## SEO
**Next.js Metadata API** - Native support for SEO, Open Graph, and Robots.txt generation.
**JSON-LD** - Structured data for search engines.

## Deployment
**Standard Next.js Build** - Compatible with Vercel, Cloudflare, or any Node.js hosting environment.
